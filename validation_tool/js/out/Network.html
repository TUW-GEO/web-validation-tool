Class: Network

Network

new Network(map, networkObject, color, visible, infoWindowEnabled, menuEnabled, networkMarkerZoom)

This Class is responsible for each Network, it handles the stations and the visibility of the correct markers (network or stations) and also the info window of the network marker.
Parameters:
Name Type Description
map google maps object google maps object the ISMN should work on
networkObject object js object from json file containing information about the network and its stations
color string hex color code of network color
visible boolean if the network should be visible from the start
infoWindowEnabled boolean if the markers should have infoWindows, only necessary for static overviews
menuEnabled boolean if a menu exists - does not for static overviews
networkMarkerZoom int zoomlevel at which the network marker should be switched for the station markers
